Yêu cầu sử dụng project:
- Microsoft Visual Studio 2008
- DevExpress 12.1 hoặc cao hơn
- DotNetBar 11.2.0.1 hoặc cao hơn
Trường hợp sử dụng version cao hơn phải convert project:
+ Start menu => DevExpress DXperience 12.x\Tools\DXperience 12.x Project Converter
+ Brown đến thư mục .\APP\SOURCE
+ Ấn Convert để cập nhật assembler cho phiên bản hiện tại
- SQL Server Enterprise/Express 2005/2008/2012 (Khuyến nghị SQL Server Express 2008)

Sử dụng chương trình:
1/ Chạy .\EXE\QuanLyDaoTao_Installer.exe để tiến hành cài đặt
2/ Click vào biểu tượng QuanLyDaoTao trên desktop
3/ Chương trình sẽ khởi động form "Tự cấu hình kết nối" cho chương trình:
  + Chọn hoặc gõ tên server
  + Nhấn Kết nối
  + Chọn database có sẵn hoặc tạo mới
  + Nhấn tạo dữ liệu mẫu cho chương trình
  + Nhấn lưu cấu hình
